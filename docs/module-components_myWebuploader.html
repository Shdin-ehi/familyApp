Module: components/myWebuploader

文件上传模块
Parameters:
Name Type Default Description
rotatable Boolean true 是否开启图片旋转,需后台根据旋转角度参数rotation处理
config Object 验证规则
uploadAccept function 详见webuploader官网,uploadAccept事件，返回true时触发uploadSuccess，false触发uploadError
Source:
  • components/myWebuploader/myWebuploader.vue, line 2
Listens to Events:
  • event:uploadSuccess 文件上传成功时触发，参数为服务器响应值
  • event:uploadComplete 文件上传完成后触发，参数为WebUploader的file对象
  • event:uploadFinished 所有文件上传结束后触发
Example
<my-webuploader :config="{
          server:'/add',
          accept: {
            title: 'Images',
            extensions: 'gif,jpg,jpeg,bmp,png',
            mimeTypes: 'image/*'
          }
        }" @uploadSuccess="uploadSuccess"></my-webuploader>
