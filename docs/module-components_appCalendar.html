Module: components/appCalendar

日历面板模块。 部分参数详细介绍见文档http://www.muse-ui.org/#/datePicker
Parameters:
Name Type Default Description
dateTimeFormat Object 时间格式化对象，见文档
firstDayOfWeek Number 1 那一天作为一个星期的开始， 默认星期一， 有些情况可能会设置成星期日(0)
initialDate Date today 初始化显示的日期
maxDate Date 可选择的最大日期
minDate Date 可选择的最小日期
mode String portrait 竖屏显示，landscape横屏显示
shouldDisableDate function 判断日期是否不可用的函数，参数为当前做判断的日期,所有返回true的为匹配日期
shouldActivedDate function 判断日期处于活动状态的函数，参数为当前做判断的日期,所有返回true的为匹配日期

Listens to Events:
  • event:selected 日期被选择时触发的事件，参数为被点击日期date
  • event:monthChange 月份变化时触发，参数为新月份第一天的date
Example
<app-calendar class="app-calendar" :maxDate="today" :shouldDisableDate="shouldDisableDate" :shouldActivedDate="shouldActivedDate" @selected="handleSelected" @monthChange="handleMonthChange"/>
