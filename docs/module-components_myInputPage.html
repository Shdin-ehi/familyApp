input输入页面模块
Parameters:
Name Type Default Description
name String input的name,若要使用验证,则必填
value String input的value，使用v-model时传入
open Boolean false 是否弹窗输入层
topTitle String 输入层标题
rules Object 验证规则
errors Object | String 匹配验证的错误
display Boolean true 是否显示当前值

<my-input-page topTitle="更改名称" v-model="itemData.name" :open="showInput.ipt4" @close="showInput.ipt4=false"  name='locateName' :rules="{required:'required', maxlength:'10',maxLength:10}" :errors="{required:'必填项',maxlength:'必须小于10个字符'}"/>
