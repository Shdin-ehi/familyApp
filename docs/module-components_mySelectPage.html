Module: components/mySelectPage

选值页面模块，支持多选
Parameters:
Name Type Default Description
name String input的name,若要使用验证,则必填
value String input的value，使用v-model时传入
open Boolean false 是否弹窗输入层
topTitle String 输入层标题
rules Object 验证规则等(绑定在输入框上)
errors Object | String 匹配验证的错误
options Array.<Object> 供选的值,label为展示值,nativeValue为表单值
multiSelect Boolean false 是否启用多选，若启用，表单值为数值，如[1,2]
display Boolean true 是否显示当前选中的显示值
Source:
  • components/mySelectPage/mySelectPage.vue, line 2
Listens to Events:
  • event:close 弹窗关闭时触发
  • event:change 表单值确认修改时触发
  • event:input 用于触发更新v-model
Example
<my-select-page topTitle="提醒时间" v-model="itemData.tipTime" :open="showInput.ipt1" @close="showInput.ipt1=false"  name="tipTime" :rules="{required:'required'}" :errors="{required:'至少选择一项'}" :options="[{label:'一年',nativeValue:'1'},{label:'两年',nativeValue:'2'},{label:'三年',nativeValue:'3'},{label:'五年',nativeValue:'4'},{label:'十年',nativeValue:'5'},{label:'永久',nativeValue:'6'}]"/>
